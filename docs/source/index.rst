=================================
MatryoshkaNet Documentation Index
=================================

Welcome to the technical documentation for **MatryoshkaNet**, a modular, private VPN management system currently under active development.

--------------
Project Status
--------------

This project is **under active development**. Features, APIs, and architecture may change frequently. Feedback and contributions are welcome but should be aligned with the core design vision.

^^^^^^^
Purpose
^^^^^^^

MatryoshkaNet is designed to provide:

* Centralized management of VPN subscriptions and access
* Fine-grained traffic limits and access control
* Integration with lightweight VPN nodes using `sing-box`
* Secure internal infrastructure without public user panels


For any questions or suggestions, please contact the maintainers or open a discussion in the GitHub repository.
